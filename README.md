# grocery

## Project setup
```
npm install
```

### dirigete a vue
```
 cd ~/Descargas/grocery-vue-project-main
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
### Ejecuta los comandos que siguen en la carpeta grocery-backend
```
Vuejs---Vuetify-JS-How-to-create-a-Responsive-E-Grocery-Website-System-Grocery-e-Commerce-Website-main/grocery-backend
```

### Cargas agente cuando se hacen cambios
```
mvn clean install
```
### Arranca el proyecto
```
npm run dev
```
